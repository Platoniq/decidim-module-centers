# Decidim::Centers

Manage your centers and scopes so the users can be authorized over them.

## Usage

Centers will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-centers"
```

And then execute:

```bash
bundle
```
